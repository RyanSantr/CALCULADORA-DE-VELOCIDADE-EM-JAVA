# 🚗 Calculadora de Velocidade Média

![Java](https://img.shields.io/badge/Java-17+-blue.svg)
![Swing](https://img.shields.io/badge/GUI-Swing-orange.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

Uma aplicação desktop desenvolvida em Java com interface gráfica Swing para calcular a velocidade média de trajetos com múltiplos trechos.

## 📋 Funcionalidades

- ✅ **Adicionar Trechos**: Registre distância (km) e tempo (min) de cada parte do percurso
- ✅ **Cálculo Automático**: Calcula a velocidade média de todo o trajeto
- ✅ **Validação de Dados**: Não permite valores negativos ou zero
- ✅ **Limpeza de Dados**: Opção para resetar todos os dados com confirmação
- ✅ **Interface Amigável**: Design limpo e intuitivo
- ✅ **Cores Personalizadas**: Interface visual agradável

## 🎯 Como Usar


## 1 **ADICIONAR O TRECHO PERCORRIDO**

Digite a distancia em Km - ex 120
Digite o tempo em minutos - ex 90
Clique em "ADICIONAR"

O Trecho será adicionado e mostradado na área de Texto.
Se não houver mais trechos, só cliclar em " CALCULAR "

# 2 **CALCULAR VELOCIDADE MEDIA**

O resultado mostrará:
 Velocidade media Total
 DIstancia Total
 Tempo Gasto

 # 3 **LIMPAR DADOS**

 Ao cliclar em limpar Dados, voce resetará todos os dados que foram adicionados
 Com isso, o sistema pedirá para você adicionar novos dados de trechos.
